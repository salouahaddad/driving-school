Le code de la route

Le code de la route est la première épreuve de l'examen du permis de conduire. Permettant de connaître toutes les règles de la conduite, l'épreuve du code est composée de quarante questions portant chacune sur une dizaine de thèmes. L’auto-école à Paris dispense des heures de cours de code pour permettre aux candidats de s’entraîner sur des séries de questions par thèmes dans un premier temps, puis, à l’examen en lui-même par la suite. Les moniteurs accompagnent chaque candidat pour suivre leur apprentissage et vérifier les acquis avant les épreuves.
